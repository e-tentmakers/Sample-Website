# Sample Church Website

A responsive church website built with HTML, CSS, and JavaScript.

## Features

- Responsive design for mobile and desktop
- Navigation with dropdown menus
- Multiple pages for church information
- Contact form integration
- Image gallery

## Setup

This website is hosted on GitHub Pages and is publicly accessible.

## Structure

- `index.html` - Homepage
- `pages/` - Subpages organized by category
- `css/` - Stylesheets
- `js/` - JavaScript files
- `Image files/` - All website images
